# Лендинг тура «Португалия 2026»

Готовый мини-репозиторий со статическим лендингом для турагента.

## Что внутри

- `index.html` — структура страницы и контент тура
- `styles.css` — стили лендинга
- `script.js` — интерактив:
  - приветственный экран
  - мобильное меню
  - таймер до старта тура
  - базовая валидация формы заявки

## Как запустить

### Вариант 1 (самый простой)

Просто откройте `index.html` в браузере.

### Вариант 2 (через локальный сервер)

Если установлен Node.js:

```bash
npx serve .
```

Если установлен Python:

```bash
python -m http.server 5500
```

После запуска откройте браузер по адресу локального сервера.
